How It Works

Get started in minutes with our simple integration process.

1

Download Files

Download the compiled JavaScript and CSS files from our repository.

2

Include Scripts

Add the script and stylesheet tags to your HTML page header.

3

Add Element

Use the <pomodoro-timer> custom element anywhere in your HTML.

4

Enjoy!

Your Pomodoro timer is ready to boost productivity on your site.

Easy Integration

Add the Pomodoro timer to your website with just a few lines of code.

<!-- Include the required files -->
<script src="dist/pomodoro-timer/browser/polyfills.js"></script>
<script src="dist/pomodoro-timer/browser/main.js"></script>
<link rel="stylesheet" href="dist/pomodoro-timer/browser/styles-EHAUOCOG.css">

<!-- Use the custom element -->
<pomodoro-timer></pomodoro-timer>
<!-- Multiple instances work independently -->
<div class="work-timer">
  <h3>Work Session</h3>
  <pomodoro-timer></pomodoro-timer>
</div>

<div class="break-timer">
  <h3>Break Session</h3>
  <pomodoro-timer></pomodoro-timer>
</div>
