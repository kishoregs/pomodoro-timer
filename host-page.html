Pomodoro Timer Widget

Angular 19 Custom Element

A modern, fully-featured Pomodoro timer built as an Angular custom element. Drop it into any web page or application with a single HTML tag.

Live Demo

Features

  • 25-minute work sessions with 5-minute breaks
  • Long breaks every 4 sessions (15 minutes)
  • Start, pause, reset, and skip functionality
  • Snooze feature (+5 minutes)
  • Customizable work and break durations
  • Audio notifications when sessions complete
  • Beautiful progress ring with animations
  • Responsive design for all devices
  • Modern glassmorphism UI design
  • Built with Angular 19 and TypeScript

How to Use

The necessary JavaScript and CSS files are dynamically loaded by a script at the end of this HTML page. Once loaded, you can use the custom element in your HTML:

<!-- Use the custom element anywhere in your HTML -->
<pomodoro-timer></pomodoro-timer>

The widget is completely self-contained and doesn't require any additional configuration. It automatically saves user preferences to localStorage.

<!-- Multiple instances work independently -->
<div class="timer-container">
  <pomodoro-timer></pomodoro-timer>
</div>

<div class="another-timer">
  <pomodoro-timer></pomodoro-timer>
</div>
